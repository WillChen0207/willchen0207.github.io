Hexo相关指令

本地测试

$ hexo clean
$ hexo generate
$ hexo server

本地服务器将启动，默认端口为4000

部署至远端服务器

$ hexo clean
$ hexo generate
$ hexo deploy

hexo相关文件将被部署至网站根目录下_config.yml文件中deploy属性规定的域名
